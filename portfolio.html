Number guessing game

Project Overview

This game was created as a part of a school project with the main purpose to learn Javascript and DOM-manipulation. The project is hosted on Github-pages.

Game description

  • A random integer between 0-100 will be generated.
  • The user have 5 guesses to guess the correct number.
  • If the guessed number is too high or too low the user will notified with a popup.
  • If the user runs out of guesses without guessing the correct number a losing message will be displayed and the user has the option to restart the game and play again.
  • When the correct number is guessed a winning message will be displayed and the user has the option to restart the game and play again.
