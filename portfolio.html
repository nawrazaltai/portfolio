Projects

Background:

The project below is my first and only IT-project (outside this portfolio page), the project itself was the last part of the DevOps Engineering program. The main concept was to create some sort of application that uses a database to store its data.

Idea:

I chose to create an application with a graphical user interface for a car parking monitor. The app is built with Python, Tkinter and CustomTkinter. For the database I chose to work with SQLite. Total amount of parking spaces for this specific parking lot is 50 spots where each registration number can't be duplicated for one single parking session. The price is free of charge for the first 60 minutes, thereafter the price is set to 0.25 SEK/minute. The application has three main funtions:

  1. Start parking
  2. View parked car status
  3. Stop parking

Main functions

The Start function:
After clicking on start parking on the main menu the client gets to the page demonstrated on the first picture below. Here the client gets asked to enter a registration number consisting of 6 characters. If the registration number is already entered in the database the client will get an error and will be asked to enter another number. If the number is formated correct and is unique, the parking session will be successfully started, shown on the second picture.
